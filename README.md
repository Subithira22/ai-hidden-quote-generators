 AI Hidden Quote Generator

# Project Overview

AI Hidden Quote Generator is a web-based application that generates meaningful quotes based on user-selected emotions. It helps users express and understand their feelings through motivational and supportive messages.

# Features

* Emotion-based quote generation
* Interactive user interface
* Multiple feelings selection
* Voice output (Speak Feeling feature)
* Simple and responsive design

# Technologies Used

* HTML
* CSS
* JavaScript

# Project Structure

* `index.html`
* `style.css`
* `script.js`

# How to Use

1. Select your feeling
2. Click on **Generate AI Quote**
3. View the generated quote
4. Use **Speak Feeling** for voice output

# Outcome

This project enhances user engagement by combining emotional intelligence with interactive web design.
